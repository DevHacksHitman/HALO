Judging Criteria

Use the Venice API to give your agents permissionless intelligence. We want builders to explore what becomes possible when agents can reason, see, speak, create, transact, and access crypto infrastructure through one API layer. Venice provides endpoints for text, image, audio, music, video, and crypto RPC access, with x402 support for agentic payments.

Possible Directions

Use Venice in a creative way across AI agents, crypto applications, onchain workflows, media generation, payments, developer tooling, or any other area where permissionless intelligence improves the product experience.

Projects that combine Venice with MetaMask, on-chain data, x402, or multiple Venice endpoints will score higher.

Qualification Requirements

To be eligible, the project must qualify for one of the three main tracks: Best x402 + 7710, Best Agent, or Best A2A Coordination.

The project should use Venice as a core part of the application

The project demo video should show Venice being used in the main flow of the application

The project should produce a meaningful AI powered output, action, or user experience through Venice

Judging Criteria

Use 1Shot API’s permissionless relayer to execute EIP-7710 transactions while paying for gas in stablecoins like USDC, USDT, USDG, MUSD, etc. 

Possible Directions

The gas abstraction relayer should pair naturally with all other technical tracks. Projects that leverage the relayer webhooks as the source for transaction status updates will score higher. You could even build your own x402 7710 facilitator on top of the 1Shot public relayer.
