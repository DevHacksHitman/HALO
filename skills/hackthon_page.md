Description

The MetaMask Smart Accounts Kit  x 1Shot API Hackathon invites developers, designers, and innovators to build next-generation agentic experience focused on permission sharing, and user experience.

About MetaMask Smart Accounts Kit

The MetaMask Smart Accounts Kit enables developers to create new experiences based on programmable account behavior and granular permission sharing. It offers a suite of contracts, libraries, and services designed for maximum composability, allowing developers to build and extend their dApps with ease.

About 1Shot API

1Shot API is the ultimate high-performance Web3 API for autonomous agents and intelligent wallets. 1Shot’s Permissionless Relayer gives you gas abstraction for EIP-7710 smart accounts with no signup, no business account, and no tier-management overhead. You can integrate directly over JSON-RPC and ship immediately. 1Shot is built for enterprise-scale traffic, so when usage spikes you don't need to upgrade plans or pre-fund infrastructure like a paymaster - 1Shot handles operational capacity so your team can focus on product growth. Install the public relayer skill into your agentic coding environment to ship fast and confidently.

About Venice AI

Venice is a privacy-first AI platform with 200+ frontier and open-source models across text, image, video, code, and audio. Enjoyed by over 3 million users, Venice brings you the full power of AI without the surveillance, supporting models from OpenAI, Anthropic, xAI, and Google, alongside Venice's own uncensored models and fully-private open-source models. Venice is accessible through its app, an OpenAI-compatible API, or permissionlessly via staking. Learn more at venice.ai.

Qualification Requirements
The project should use the Metamask Smart Accounts or Advanced Permissions. MetaMask Smart Accounts are signer agnostic, so you can use any wallet provider of your choice i.e. MetaMask extension, MetaMask Embedded Wallets, Dynamic or Privy.

Advanced Permissions (ERC-7715) are fine-grained permissions dApps can request from users directly via the MetaMask browser extension. Advanced Permissions allow you to perform executions on the behalf of MetaMask users.

To integrate MetaMask Smart Accounts or Advanced Permissions use the MetaMask Smart Accounts Kit.

The project demo video should have a working MetaMask Smart Accounts Kit integration(either Smart Accounts or Advanced Permissions) in the main flow of the application.

The judges can decide not to reward a track prize if no project meets the standards set by them internally, apart from the basic submission criteria.
